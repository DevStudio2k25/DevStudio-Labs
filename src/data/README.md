# Data Structure

This folder keeps project content embedded in the codebase.

## Folder

- `projects`: real project JSON files shown on the website.

## Project File Shape

Each project is one JSON file with:

- `id`
- `slug`
- `audience`
- `title`
- `category`
- `status`
- `featured`
- `summary`
- `overview`
- `problem`
- `solution`
- `techStack`
- `features`
- `deliverables`
- `media`
- `links`
- `accent`
- `tags`
- `timeline`

Use `src/data/projects.js` to import mixed, featured, student, or freelancer project lists.
